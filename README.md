Ejercicio Práctico – Manejo de Promesas en JavaScript
Descripción

Las promesas en JavaScript permiten manejar operaciones asincrónicas de forma clara y estructurada. Son fundamentales para trabajar con procesos que no se ejecutan de inmediato, como validaciones, consultas a servidores o tareas que implican tiempos de espera.

En este ejercicio se trabaja con promesas simuladas, enfocadas en comprender su funcionamiento básico sin depender de servicios externos.

Objetivo

El objetivo de esta actividad es:

Comprender cómo crear promesas en JavaScript.

Entender cuándo y cómo se resuelven (resolve) o se rechazan (reject).

Analizar el comportamiento de operaciones asincrónicas.

Interpretar correctamente los mensajes que aparecen en la consola según distintos escenarios de ejecución.

Contenidos abordados

Uso del objeto Promise

Estados de una promesa: pendiente, resuelta y rechazada

Manejo de resultados con .then()

Manejo de errores con .catch()

Simulación de retrasos con setTimeout

Flujo de ejecución asincrónico
