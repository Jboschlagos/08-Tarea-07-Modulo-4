Manejo de Promesas en JavaScript
Introducción / Objetivo
Las promesas en JavaScript permiten manejar operaciones asincrónicas de manera clara y estructurada. 
En esta actividad trabajarás con promesas aplicadas a validaciones, consultas simuladas y ejecución de procesos que pueden generar retrasos. 
El objetivo es comprender cómo crear, resolver y rechazar promesas, así como interpretar los mensajes que aparecen en consola en distintos escenarios.
